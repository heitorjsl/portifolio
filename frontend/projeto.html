Conheça o projeto

A Calculadora Penal é uma ferramenta desenvolvida para auxiliar profissionais do Direito no cálculo de penas, prazos e benefícios previstos na legislação penal brasileira. O objetivo do projeto é oferecer uma solução prática, rápida e precisa, que contribua para a análise processual e a tomada de decisões jurídicas.

O sistema foi criado com base nos princípios da clareza, eficiência e confiabilidade, utilizando a linguagem Kotlin para garantir desempenho e segurança.

Desenvolvido por Beatriz e Heitor, durante a disciplina PAE CC3505-Desenvolvimento de Aplicativos em Linguagem Kotlin, o projeto reflete o compromisso com a inovação tecnológica aplicada ao campo jurídico.
